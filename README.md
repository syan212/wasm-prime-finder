# Prime finder

Simple website for finding primes made using WASM.

[View online](https://syan212.github.io/wasm-prime-finder/)

## How to run locally

Ensure cargo is installed and run
```shell
cargo install wasm-pack # Install wasm-pack required for compiling to was
wasm-pack build --target web # Compile
```
and then open `index.html` in a web browser.
